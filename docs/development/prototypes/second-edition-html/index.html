System Overview

This prototype documentation represents the corrected and standardized version of the Cosmos Engine RPG skill system. All skill trees now follow the unified 7-rank progression with proper mastery tier organization.

Key System Changes:
  • 7-Rank Progression: Skills now have exactly 7 ranks at levels 1, 2, 3, 4, 6, 8, 10
  • 4 Mastery Tiers: Basic (1) → Journeyman (2-3) → Master (4-6) → Grandmaster (7-10)
  • Universal Tag System: 4 tags per rank creating synergistic combinations
  • 18 Core Skills: Added Performance, Craft, and Commerce skills for 95%+ archetype coverage

System Mechanics

Skill Progression Pattern

7 Total Skills per Tree:

  • Rank 1: Basic - Foundation skill
  • Rank 2: Journeyman - Professional development
  • Rank 3: Journeyman - Advanced techniques
  • Rank 4: Master - Expert capability
  • Rank 6: Master - Scale transition
  • Rank 8: Grandmaster - Legendary ability
  • Rank 10: Grandmaster - Transcendent mastery

Tag System Integration

Each skill rank offers 4 tag choices that create synergistic combinations:

  • Single Tags: Basic bonuses (+1 die)
  • Double Synergies: Two related tags (+2 dice)
  • Triple Synergies: Three harmonizing tags (+3 dice)
  • Cross-Skill Synergies: Tags from different skills (+4 dice)
  • Ultimate Combinations: Archetype-level mastery (+5 dice)
