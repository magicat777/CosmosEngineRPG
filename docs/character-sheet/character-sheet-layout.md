## DESIGN NOTES

### Key Features
1. **Four-page layout** suitable for double-sided printing
2. **Clear sections** for all core mechanics
3. **Scale tracking** integrated into skills
4. **Health state checkboxes** for easy tracking
5. **Session log** for advancement points
6. **Equipment condition** tracking built-in
7. **Combat reference** on same page as weapons

### Digital Version Features
- Fillable form fields for all entries
- Auto-calculation for derived statistics
- Dropdown menus for skill scales
- Expandable sections for equipment/notes
- Print-friendly black & white option

### Accessibility Considerations
- High contrast borders
- Clear section headers
- Consistent layout patterns
- Space for large handwriting
- No tiny text boxes

This layout provides everything needed for play while maintaining clarity and usability at the table.
