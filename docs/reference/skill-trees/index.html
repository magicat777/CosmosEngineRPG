Skill System Overview

7-Rank Progression

  • Rank 1: Basic (Foundation)
  • Ranks 2-3: Journeyman
  • Ranks 4-6: Master
  • Ranks 7-10: Grandmaster

Skills available at ranks: 1, 2, 3, 4, 6, 8, 10

Universal Tag System

  • 4 Tags per Rank: Choose one from each rank
  • Synergy Bonuses: Tags combine for extra dice
  • Cross-Skill Synergies: Tags from different skills
  • Archetype Definition: Tag combinations define roles

Total possible: 28 tags per character at max rank
