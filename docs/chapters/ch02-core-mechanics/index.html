1. Quick Start: The 2d10 System

The Universal Formula

2d10 + Attribute Modifier + Equipment + Tag Dice vs Target Number (TN)

Roll two ten-sided dice, add modifiers, compare to Target Number. That's it.

üé≤ How Bonus Dice Work

When you gain bonus dice from tags, you roll 2d10 + bonus dice, then use the two highest results.

  • +1 die = Roll 3d10, keep highest 2
  • +2 dice = Roll 4d10, keep highest 2
  • +3 dice = Roll 5d10, keep highest 2

This creates dramatic swings while keeping math simple.

Basic Probability

2d10 Result Probability Occurrence
2 (Critical Failure) 1% 1 in 100
8-14 (Core Range) 72% ~3 in 4
11 (Average) 9% Most common
20 (Critical Success) 1% 1 in 100

When to Roll

Only roll when:

  • The outcome is uncertain
  • Success and failure are both interesting
  • There's meaningful risk or time pressure

Don't roll for routine tasks competent characters should handle.

When to Use Modifiers vs Raw Scores

  • Use Modifiers for: All skill checks, attack rolls, damage bonuses (melee), initiative, and movement speed
  • Use Raw Attribute Scores for: Defense calculations (10 + Coordination), derived values (Health = 10 + Might), and meeting prerequisites

3. Resolution Process

Complete Resolution Example

Scenario: Lt. Chen attempts to hack a security door while under fire.

  1. Intent: "I hack the door's lock mechanism"
  2. GM Sets TN: Hard task (TN 17) due to combat stress
  3. Build Dice Pool:
    • Base: 2d10
    • Intellect 4 (+1 modifier)
    • Technology Tags: [Military Tech] +1 die, [Quick Thinking] +1 die
    • Equipment: Hacking Kit +2
  4. Roll: 4d10 (base + 2 tag dice) = 7, 9, 4, 10 ‚Üí Keep 9, 10 = 19
  5. Total: 19 + 1 + 2 = 22 vs TN 17
  6. Result: Success by 5! Door opens immediately, surprising enemies.

Success Margins

Margin Result Description
Failure by 10+ Catastrophic Things get much worse
Failure by 1-9 Simple Failure Doesn't work as intended
Success by 0-4 Basic Success Achieves primary goal
Success by 5-9 Strong Success Achieves goal with style
Success by 10+ Exceptional Bonus effects apply

Critical Results

Natural 2 (Snake Eyes)

Automatic failure regardless of modifiers. Roll again - if second roll also fails, catastrophic consequences occur.

Natural 20 (Perfect Roll)

Automatic success with exceptional results. Achieve primary goal plus beneficial side effects.

4. Modifiers Reference

Tag Bonus Summary

Individual Tag Limits

  • Most tags provide +1 die
  • Specialized tags may give +2 dice
  • Legendary tags grant up to +3 dice

Total Bonus Limits

  • Maximum +5 dice from all tag sources combined
  • This represents rolling 7d10 keep 2 highest
  • Provides ~85% chance to roll 15+

Equipment vs Tags

Type Bonus Format Stacking Example
Equipment Flat modifier (+1 to +3) Only highest applies Smart Rifle +2
Tags Bonus dice (+1 to +3 dice) All relevant tags stack [Sniper] +1 die

Key Difference: Equipment provides consistent bonuses. Tags create dramatic possibilities.

Penalty Cap: -6

No matter how many negative modifiers apply, the total penalty cannot exceed -6.

  • Wounded: -2
  • Poor visibility: -2
  • Difficult terrain: -2
  • Total: -6 (not -8)

Bonus Cap: +8

No matter how many positive modifiers apply, the total bonus cannot exceed +8.

  • Skill expert: +5
  • Superior equipment: +3
  • Ideal conditions: +2
  • Total: +8 (not +10)

Important: These caps apply to the standard resolution method. When using tag synergies for bonus dice, these modifier caps are replaced by the dice pool mechanics (max 7d10 keep 2).
