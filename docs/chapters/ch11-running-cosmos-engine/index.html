Chapter 11: Running Cosmos Engine

Game Master guidance for creating memorable cosmic adventures using the Universal Tag System to enhance storytelling and mechanical resolution.

11.1 The GM's Role with Tags

As GM, you're the conductor of a cosmic orchestra where player tags create the melody and environmental tags provide the harmony. Your role is to recognize tag synergies and weave them into compelling narratives.

Tag-Driven Storytelling

Spotlight Character Tags

  • Create situations where each PC's tags shine
  • Design encounters that reward diverse approaches
  • Let tags guide NPC reactions

Environmental Tags

  • Every location should have 2-3 descriptive tags
  • Tags create opportunities and obstacles
  • Let players suggest environmental tags

11.2 Campaign Scales

Cosmos Engine naturally scales from personal stories to galactic epics. Use the tag system to maintain coherence across scales.

Personal Scale (Ranks 1-3)

Individual adventures, local problems, getting to know each other

System Scale (Ranks 4-6)

Multi-planet adventures, system-wide threats, reputation building

Galactic Scale (Ranks 7-9)

Sector-spanning adventures, galactic politics, legendary status

Cosmic Scale (Rank 10)

Reality-altering adventures, universe-threatening events

11.3 Adventure Design Philosophy

Core Principles

  • Tags Tell Stories: Every tag should suggest narrative possibilities
  • Multiple Solutions: Design problems that can be solved with different tag combinations
  • Evolving Tags: Let character and equipment tags grow through play
  • Synergy Rewards: Recognize and reward creative tag combinations
  • Scale Appropriately: Match challenge scope to character advancement
