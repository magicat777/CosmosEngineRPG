Equipment Tags

How Equipment Tags Work

Equipment tags create synergies with character tags, providing +1 to +5 dice when relevant combinations occur.

Equipment Tags + Character Tags = Bonus Dice

Equipment Tag Types

Tag Type Description Examples
Quality Tags Craftsmanship level [Basic], [Advanced], [Master-Crafted], [Legendary]
Function Tags What it does [Accurate], [Silent], [Devastating], [Precise]
Tech Tags Technology type [Mechanical], [Electronic], [Quantum], [Psionic]
Condition Tags Current state [Worn], [Reliable], [Pristine], [Jury-Rigged]

Tag Synergy Example

Scenario: Sniper shot with high-tech rifle

  • Character: [Precise] + [Patient] = +2 dice
  • Weapon: [Accurate] + [Advanced] = +2 dice
  • Total: +4 dice (roll 6d10, keep highest 2)

Modifications

Equipment can be modified to add new tags:

  • Weapon Mods: Scopes (+[Accurate]), Silencers (+[Silent]), Smart-links (+[Advanced])
  • Armor Mods: Reinforcement (+1 DR), Stealth coating (+[Concealed]), Sensors (+[Aware])
  • Cost: 25-50% of base item cost per modification

Cybernetics

Enhancement augmentations providing permanent bonuses:

  • Neural Interface: +1 to Technology rolls, 5,000cr [Comfortable]
  • Enhanced Reflexes: +1 Initiative, 8,000cr [Comfortable]
  • Cyber-Eyes: +2 to Perception rolls, 6,000cr [Comfortable]
  • Reinforced Skeleton: +2 Health, 15,000cr [Wealthy]

Installation requires Medical TN 17 and compatible facilities

Acquisition During Play

  • Purchase: Pay credit cost if available
  • Wealth Access: Items at/below your wealth level are readily available
  • Mission Rewards: Gain equipment through play regardless of wealth
  • Modification: Improve existing gear with time and resources

Quick Reference

EQUIPMENT QUICK REFERENCE

Equipment Tags

  • [Basic] - Standard quality
  • [Advanced] - High-tech
  • [Master-Crafted] - Exceptional
  • [Accurate] - +hit chance
  • [Silent] - No noise
  • [Devastating] - High damage
  • [Rapid] - Fast attacks
  • [Reliable] - Never jams
  • [Concealed] - Hidden
  • [Military] - Tactical

Tag Synergy Formula

Character Tags + Equipment Tags = Bonus Dice (max +5)

Basic Synergy
1-2 matching tags
+1 to +2 dice
Strong Synergy
3-4 matching tags
+3 to +4 dice
Perfect Synergy
5+ matching tags
+5 dice (maximum)
