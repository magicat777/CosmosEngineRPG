Chapter 5: Actions and Skills

Cosmos Engine features 18 core skills that scale from personal actions to galactic operations. Each skill rank grants one tag that defines how you approach challenges.

Chapter Contents

  • How Skills Work
  • The 18 Core Skills
  • Quick Reference

How Skills Work

Basic Resolution

2d10 + Attribute + Skill + Tag Bonuses vs Target Number

When to Roll: Only when outcome is uncertain and both success and failure are interesting.

Attribute Pairing: GM selects most relevant attribute for the specific action:

  • Combat + Might = Melee attacks
  • Combat + Coordination = Ranged attacks
  • Combat + Intellect = Tactical planning

Tag System Reminder

Tags represent HOW you do things, providing bonus dice when relevant.

  • Each skill rank grants one tag
  • Relevant tags provide +1 die each
  • Maximum +5 dice from all tags combined

See Chapter 2 for complete tag rules and synergy mechanics.

Scale and Skills

Skills work at all scales. As you advance, tags evolve in meaning:

Example: Combat Across Scales

  • Personal: Firefight in a space station corridor
  • Vehicle: Tank battle on planetary surface
  • Starship: Destroyer engagement in asteroid field
  • Capital: Fleet admiral coordinating sector assault

Same skill, same mechanics, different narrative scope.

The 18 Core Skills

Combat & Physical Skills

COMBAT - The Art of Warfare

Primary Attributes: Might (melee), Coordination (ranged), Awareness (tactics)

Core Concept: Personal combat to fleet warfare using consistent mechanics.

Scale Evolution: Dueling → Squad tactics → Ship gunnery → Fleet command

Common Uses: Attack rolls, defensive maneuvers, tactical assessment, weapon mastery

Full progression available in Cosmos Engine Skills Compendium

ATHLETICS - Physical Excellence

Primary Attributes: Might (power), Coordination (agility), Resolve (endurance)

Core Concept: Movement, acrobatics, and physical feats across all environments.

Scale Evolution: Personal parkour → Vehicle stunts → EVA operations → Fleet formations

Common Uses: Climbing, jumping, swimming, zero-g maneuvering, endurance

STEALTH - The Silent Art

Primary Attributes: Coordination (physical), Awareness (timing), Intellect (planning)

Core Concept: Avoiding detection through physical and electronic means.

Scale Evolution: Personal infiltration → Vehicle cloaking → Ship silent running → Fleet concealment

Common Uses: Sneaking, hiding, shadowing, electronic warfare countermeasures

Technical & Mental Skills

TECHNOLOGY - Digital Mastery

Primary Attributes: Intellect (programming), Coordination (hardware), Awareness (diagnostics)

Core Concept: Understanding and manipulating technology from personal to planetary scale.

Scale Evolution: Device hacking → System intrusion → Ship AI → Planetary networks

Common Uses: Hacking, repairs, programming, system analysis, cybersecurity

MEDICAL - Life Sciences

Primary Attributes: Intellect (diagnosis), Coordination (surgery), Presence (bedside manner)

Core Concept: Healing and biological sciences across multiple species.

Scale Evolution: First aid → Surgery → Epidemiology → Planetary health systems

Common Uses: Treatment, surgery, disease control, xenobiology, enhancement

KNOWLEDGE - Academic Mastery

Primary Attributes: Intellect (recall), Awareness (research), Resolve (study)

Core Concept: Academic knowledge and research capabilities.

Scale Evolution: Personal expertise → Library research → Database mining → Galactic archives

Common Uses: History, science, cultures, languages, investigation

CRAFT - Creation and Repair

Primary Attributes: Coordination (precision), Intellect (design), Might (construction)

Core Concept: Building, modifying, and repairing objects of all scales.

Scale Evolution: Personal items → Vehicles → Starships → Space stations

Common Uses: Construction, modification, jury-rigging, artistic creation

PILOT - Vehicle Operation

Primary Attributes: Coordination (control), Awareness (navigation), Intellect (systems)

Core Concept: Operating vehicles from ground cars to capital ships.

Scale Evolution: Ground vehicles → Aircraft → Starships → Fleet coordination

Common Uses: Maneuvering, navigation, evasion, formation flying

Social & Support Skills

PERSUASION - The Art of Influence

Primary Attributes: Presence (charm), Intellect (logic), Resolve (persistence)

Core Concept: Convincing others through charm, logic, or persistence.

Scale Evolution: Individual → Small groups → Organizations → Planetary populations

Common Uses: Negotiation, diplomacy, motivation, public speaking

DECEPTION - Misdirection Mastery

Primary Attributes: Presence (acting), Intellect (planning), Coordination (sleight of hand)

Core Concept: Misleading others through lies, disguise, or misdirection.

Scale Evolution: White lies → Complex cons → Organizational fronts → Sector-wide propaganda

Common Uses: Lying, disguise, forgery, false identities, misinformation

INTIMIDATION - Fear as a Tool

Primary Attributes: Presence (menace), Might (physical), Resolve (will)

Core Concept: Using fear to influence behavior.

Scale Evolution: Personal threats → Gang intimidation → Military deterrence → Planetary blockades

Common Uses: Threats, interrogation, crowd control, psychological warfare

COMMAND - Leadership Excellence

Primary Attributes: Presence (authority), Intellect (tactics), Resolve (decisiveness)

Core Concept: Leading others effectively in peace and war.

Scale Evolution: Small team → Company → Ship crew → Fleet admiral

Common Uses: Giving orders, maintaining morale, tactical coordination, inspiration

PERFORMANCE - Artistic Expression

Primary Attributes: Presence (stage), Coordination (instruments), Awareness (audience)

Core Concept: Entertainment and artistic expression.

Scale Evolution: Street performance → Venues → Broadcast → Galactic tours

Common Uses: Music, acting, public speaking, distraction, morale

COMMERCE - Economic Mastery

Primary Attributes: Intellect (analysis), Presence (negotiation), Awareness (opportunities)

Core Concept: Understanding and manipulating economic systems.

Scale Evolution: Personal trades → Business deals → Corporate mergers → Sector economics

Common Uses: Trading, valuation, market analysis, resource management

PERCEPTION - Sensory Acuity

Primary Attributes: Awareness (primary), Intellect (analysis), Resolve (focus)

Core Concept: Noticing details others miss.

Scale Evolution: Personal observation → Area surveillance → Ship sensors → Fleet intelligence

Common Uses: Spot hidden, hear faint sounds, notice patterns, sensor operation

SURVIVAL - Environmental Adaptation

Primary Attributes: Resolve (endurance), Awareness (dangers), Intellect (planning)

Core Concept: Thriving in hostile environments.

Scale Evolution: Personal survival → Group resources → Colony management → Terraforming

Common Uses: Finding resources, shelter construction, hazard navigation, adaptation

DISCIPLINE - Mental Fortitude

Primary Attributes: Resolve (primary), Presence (confidence), Intellect (focus)

Core Concept: Mental resilience and self-control.

Scale Evolution: Personal focus → Team morale → Organizational culture → Societal order

Common Uses: Resist fear, maintain focus, meditation, resist mental effects
