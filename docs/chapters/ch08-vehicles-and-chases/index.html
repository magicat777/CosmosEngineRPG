Vehicle Basics & Tags

How Vehicle Tags Work

Vehicle tags create synergies with pilot tags, providing +1 to +5 dice when combinations occur.

Pilot Tags + Vehicle Tags = Bonus Dice

Vehicle Tag Types

Category Tags Effect
Performance [Fast], [Agile], [Stable], [Durable] Handling modifiers
Capability [Armed], [Shielded], [Jump], [Atmospheric] Equipment/abilities
Personality [Loyal], [Temperamental], [Proud], [Awakened] Pilot synergies
Design [Sleek], [Rugged], [Luxury], [Military] Social/environmental

Synergy Example

Scenario: High-speed chase through city traffic

  • Pilot: [Daredevil] + [Fast] = +2 dice
  • Vehicle: [Fast] + [Agile] = +2 dice
  • Total: +4 dice (roll 6d10, keep highest 2)

Chase Mechanics

Chase Structure

  1. Set Scene: Establish positions, goals, and terrain
  2. Contest Rolls: Opposed pilot + vehicle rolls each round
  3. Apply Results: Success/failure determines position and complications

Chase Complications

Failed rolls or GM decisions introduce complications:

Type Examples Effect
Traffic Civilian vehicles, construction zones +2 TN or detour required
Vehicle Problems Overheating, system malfunction -1 die until repaired
Environmental Weather, terrain hazards Tag bonuses may not apply
Third Parties Additional pursuers, authorities Multiple opponents

Example Chase

Setup: Smuggler ([Daredevil] + [Lucky]) in [Fast] [Agile] hoverbike vs Corporate security ([Military] + [Precise]) in [Armed] [Durable] patrol craft

Round 1: Smuggler threads plasma cutters (+4 dice), Security shoots while pursuing

Complication: Mining equipment activates, creating moving obstacles

Resolution: Desperate leap through blast doors - all tags apply for +4 dice!

Vehicle Combat

Attack Resolution

  1. Roll Attack: Pilot + weapon skill + vehicle tags vs target's Defense
  2. Apply Modifiers: Speed/range/cover adjustments
  3. Roll Damage: Weapon damage if hit connects
  4. Mark Damage: Apply to vehicle condition track

Vehicle Damage Track

Condition Effects Examples
Light Damage -1 to one system type Scratched hull, minor glitches
Moderate Damage -2 to affected systems System failures, performance loss
Critical Damage Major system failure Risk of shutdown, dangerous conditions
Destroyed Vehicle inoperable Explosion, complete system failure

Scale Interactions

Vehicle weapons follow scale conversion rules:

  • Personal vs Vehicle: Personal weapons deal -2 dice damage
  • Vehicle vs Personal: Vehicle weapons deal ×10 damage to individuals
  • Same Scale: Normal damage resolution

Acquisition Notes:

  • Items at/below your wealth level are readily available
  • Higher tier items require special arrangements or mission rewards
  • Military vehicles typically require clearance or black market
  • Modifications add 25-50% to base cost

Quick Reference

VEHICLE OPERATION QUICK REFERENCE

Vehicle Operation Formula

2d10 + Pilot Skill + Pilot Tags + Vehicle Tags

Tag Synergy Limits

  • Basic synergy: +1 to +2 dice
  • Strong synergy: +3 to +4 dice
  • Perfect synergy: +5 dice (maximum)
  • Tag conflicts: -1 to -2 dice penalty
