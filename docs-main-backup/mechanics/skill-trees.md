# Cosmos Engine RPG - Comprehensive Skill Trees

## Overview

Each skill in Cosmos Engine branches into specialized paths, allowing characters to develop unique expertise. Skills progress through four stages, with specialization options at each level:

- **Novice (0-2)**: Basic competency
- **Proficient (3-5)**: Professional level
- **Expert (6-8)**: Master practitioner
- **Legendary (9-10)**: Galaxy-renowned specialist

## Implementation Notes

### Specialization Mechanics

1. **Acquiring Specializations**:
   - Choose path at skill level 1
   - Gain specialization at levels 2, 5, 8, and 10
   - Each specialization provides unique abilities
   - Cannot change path once chosen

2. **Specialization Benefits**:
   - Mechanical bonuses (+2 to specific uses)
   - Unique abilities unavailable otherwise
   - Narrative permissions (do impossible things)
   - Reduced difficulty for specialized tasks

3. **Cross-Specialization**:
   - Some high-level abilities require multiple skills
   - Synergy bonuses for related specializations
   - Prestige specializations combining paths

4. **Scale Integration**:
   - Personal scale: All specializations work normally
   - Vehicle scale: Some specializations adapt (Pilot, Combat)
   - Starship scale: Tech and Command specializations crucial
   - Capital scale: Leadership and coordination vital

### Character Advancement

**Experience Costs**:
- Skill 0→1: 5 XP (Choose path)
- Skill 1→2: 10 XP (First specialization)
- Skill 2→3: 15 XP
- Skill 3→4: 20 XP
- Skill 4→5: 25 XP (Second specialization)
- Skill 5→6: 30 XP
- Skill 6→7: 35 XP
- Skill 7→8: 40 XP (Third specialization)

- **Proficient**: Achieve results beyond normal limits
- **Expert**: Break conventional rules of the discipline
- **Legendary**: Reshape reality within their domain
